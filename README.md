## 🗂 File nella cartella

| File | Cosa contiene |
|---|---|
| `index.html` | Tutta la struttura: porta di Moria + pagina 1 + pagina 2 dell'invito |
| `style.css` | Tutti gli stili grafici |
| `script.js` | Animazione stelle, apertura porta, auto-open |

**Non devi rinominare nulla.** Carica tutti e tre i file così come sono.

---

## ✏️ Modifiche future

Tutto quello che potresti voler cambiare si trova in **`index.html`**:

| Cosa cambiare | Dove cercarlo nel file |
|---|---|
| Nomi sposi | `class="name-one"` e `class="name-two"` |
| Data cerimonia | `class="month-text"` e `class="d-num"` |
| Orario | `Ore 16.00 Cerimonia` nel testo |
| Luogo | `class="location-name"` e `class="address-text"` |
| Data conferma | `il 15 Settembre` nel testo |
| IBAN | `class="iban-code"` e `class="iban-name"` |
| Quote orari timeline | I testi `ORE 17.00`, `ORE 22.30`, ecc. |

## 🚀 Mettere online su GitHub Pages (gratis)

1. Vai su [github.com](https://github.com) → crea account gratuito (se non ce l'hai)
2. Clicca **"New repository"** → chiamalo ad es. `matrimonio`
3. Carica i 3 file (`index.html`, `style.css`, `script.js`)
4. Vai su **Settings → Pages → Branch: main → / (root) → Save**
5. Dopo 1-2 minuti il sito è online su:
   `https://tuonome.github.io/matrimonio`

Quel link puoi mandarlo direttamente agli invitati!
